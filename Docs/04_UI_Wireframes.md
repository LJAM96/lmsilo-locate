## Accessibility Features

### Keyboard Navigation

- **Tab**: Move between panels and controls
- **Arrow Keys**: Navigate within image grid
- **Space/Enter**: Select/activate items
- **Ctrl+A**: Select all images
- **Delete**: Remove selected images
- **F11**: Toggle fullscreen

### Screen Reader Support

- All buttons have aria-labels
- Predictions announced as "Rank 1, Tokyo Japan, confidence 85%"
- Status changes announced (e.g., "Processing complete")
- Map pins have descriptive alt text

### High Contrast Mode

- Increase all border widths to 2px
- Use solid colors (no gradients)
- Ensure 7:1 contrast ratio for text

## Implementation Notes

### XAML Structure

```xml
<Page>
    <Grid>
        <Grid.ColumnDefinitions>
            <ColumnDefinition Width="300"/>   <!-- Left Panel -->
            <ColumnDefinition Width="*"/>     <!-- Center Panel -->
            <ColumnDefinition Width="400"/>   <!-- Right Panel -->
        </Grid.ColumnDefinitions>

        <!-- Left: Image Queue -->
        <Grid Grid.Column="0" Background="{ThemeResource CardBackgroundFillColorDefaultBrush}">
            <!-- Content -->
        </Grid>

        <!-- Center: Map/Globe -->
        <Grid Grid.Column="1" Background="{ThemeResource ApplicationPageBackgroundThemeBrush}">
            <!-- WebView2 or CanvasControl -->
        </Grid>

        <!-- Right: Results -->
        <Grid Grid.Column="2" Background="{ThemeResource CardBackgroundFillColorDefaultBrush}">
            <!-- Content -->
        </Grid>
    </Grid>
</Page>
```

### Performance Considerations

- Virtualize image queue (ItemsRepeater instead of ItemsControl)
- Lazy-load thumbnails on scroll
- Debounce search/filter inputs (300ms delay)
- Throttle map updates to 60 FPS
- Use incremental loading for large result sets
